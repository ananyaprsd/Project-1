# Play Tennis Prediction (Naive Bayes)

This project predicts whether to **Play Tennis** or **Not** based on weather conditions using the **Naive Bayes classifier**.

## Overview
- Uses the classic **PlayTennis.csv** dataset.
- Applies **one-hot encoding** to handle categorical data.
- Trains a **Gaussian Naive Bayes model**.
- Makes predictions for new input conditions.

## Requirements
```bash
pip install numpy pandas scikit-learn seaborn

How to Run

Place PlayTennis.csv in the same folder.

Run the script:

python play_tennis_naive_bayes.py

Output

Displays encoded dataset.

Trains the Naive Bayes model.

Prints predictions for given test cases.
